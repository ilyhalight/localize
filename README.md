Some utils for quick and easy localization

## Usage

To use tui utility run:

```bash
bunx @toil/localize-tui
```

## Build

To build, you must have:

- [Bun](https://bun.sh/)

Don't forget to install the dependencies:

```bash
bun install
```

Start building:

```bash
bun build:all
```
